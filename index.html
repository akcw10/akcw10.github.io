CSS Lessons

CSS Week 1 Lesson

Colors

Ways to define colors

  • English!, example: red, green, beige, yellow, etc..
  • Hexadecimal, example: #000000, #ffffff, #4d55ca, etc..
  • RGB (with/without alpha transparency, Example: rgb(1,1,1), rgba(0.5,0,8,1,0.5), rgba(1,0,1,0.7)
Text Styling
  • font-family: arial, helvetica, etc.. (remember browser ability)
  • fancy font use font face syntax
  • font-style: normal, italic, oblique
  • font-variant: normal, small-caps
  • font-size: (small, medium, large), use px, use percentage..
  • text-align: left, right, center, justify
  • text-transform:upper, lowercase, capitalize
  • line-height: (margin for each line), 50%, 200%, etc..
Display and Visibility
  • display: inline, block, inline-block, none
  • Float: left, right
  • clear: left, right, both
  • overflow: visible, hidden, scroll, auto
  • display: table (table-cell), grid, flexbox (probably not going work on certain browser)
  • visibility: visible, hidden, collapsed

CSS Week 2 Lesson

Box Model

  • border

    Style: solid, dotted, dashed, grooved, inset, outset, ridged, hidden.
    width: use px or (thin, medium, large/thick), can specify each sides width.
    color: as usual
  • Margin (space outside border): can be positive or negative values
  • Padding (space inside border): always positive values
  • Remember! Height and Width of padding and margin is additive!
  • overflow: visible, hidden, scroll, auto
  • display: table (table-cell), grid, flexbox (probably not going work on certain browser)
  • visibility: visible, hidden, collapsed
Styling Link and List

Link

  • Have text-decoration: none; to remove underline
  • Have States (with precedence) --> link, visited, hover, focus, active
  • think how to use link semantically (diferentiate with button or etc..)

List

  • list-style-type: type of markers or values for ordered number
  • list-style-image: using image as your markers
  • list-style-position: positioning your marker
CSS Selectors

what you see below is just a summary, for detailed explanation you can visit learn.shayhowe.com on "complex selectors" page

  • Common Selectors 
    								
	h1 {...} (tag type selector)
	.tagline {...} (class selector)
	#intro {...} (id selector)
  • Child Selectors 
    								
	article h1 {...} (Descendant selector)
	article > p {...} (Direct Child selector)
	#intro {...} (id selector)
  • Sibling Selectors 
    								
	h1 ~ p {...} (General Sibling selector)
	h1 + p {...} (Adjacent Sibling selector)
  • Attribute Selectors 
    								
	a[target] {...} (Attribute Present selector)
	a[href="http://google.com/"] {...} (Attribute Equals selector)
	a[href*="login"] {...} (Attribute Contains Selector)
	a[href^="https://"] (Attribute Begins With Selector)
	a[href$=".pdf"] (Attribute Ends With Selector)
	a[rel~="tag"] (Attribute Spaced Selector)
	a[lang|="en"] (Attribute Hyphenated Selector)

  • Pseudo-elements (Will be explained later..)
