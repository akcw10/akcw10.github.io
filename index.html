Colors

Ways to define colors

  • English!, example: red, green, beige, yellow, etc..
  • Hexadecimal, example: #000000, #ffffff, #4d55ca, etc..
  • RGB (with/without alpha transparency, Example: rgb(1,1,1), rgba(0.5,0,8,1,0.5), rgba(1,0,1,0.7)
Text Styling
  • font-family: arial, helvetica, etc.. (remember browser ability)
  • fancy font use font face syntax
  • font-style: normal, italic, oblique
  • font-variant: normal, small-caps
  • font-size: (small, medium, large), use px, use percentage..
  • text-align: left, right, center, justify
  • line-height: (margin for each line), 50%, 200%, etc..
